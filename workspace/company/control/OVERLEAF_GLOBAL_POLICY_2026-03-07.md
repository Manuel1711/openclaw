# Overleaf Global Policy (All Bots)

Status: ACTIVE
Owner: Manuel
Date: 2026-03-07

## Mandatory rules
1. Default Overleaf access uses OpenClaw dedicated browser (`profile=openclaw`).
2. Overleaf read/analysis is allowed without extra approval.
3. Any edit on Overleaf requires explicit Manuel pre-approval.
4. Without explicit approval, bots must remain read-only.
5. Reports should clearly state whether actions were read-only or edit-approved.

## Edit actions covered by approval rule
- file content edits
- file create/delete/rename/move
- upload/replace assets
- structural compile-oriented changes

## Enforcement intent
- consistency across specialists/chiefs
- reduction of accidental edits
- clear human-in-the-loop control
