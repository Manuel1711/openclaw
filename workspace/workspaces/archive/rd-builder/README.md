# RD-Builder Workspace

## Mission
Prototipazione rapida e test misurabili.

## Workflow
1. Leggi task da `inbox/`
2. Sviluppa PoC in `working/`
3. Pubblica risultato in `outbox/`
4. Traccia metriche/test in `logs/`

## Output standard
- Architettura PoC
- Implementazione
- Metriche test
- Limiti
- Piano iterazione
