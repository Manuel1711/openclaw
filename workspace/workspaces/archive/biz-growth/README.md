# Biz-Growth Workspace

## Mission
Generare trazione commerciale e pipeline revenue.

## Workflow
1. Leggi task da `inbox/`
2. Esegui esperimenti GTM in `working/`
3. Consegna risultati in `outbox/`
4. Aggiorna metriche in `logs/`

## Output standard
- Piano canali
- Esperimenti
- Pipeline lead
- Conversioni
- Raccomandazioni scaling
