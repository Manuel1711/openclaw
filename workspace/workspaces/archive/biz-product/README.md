# Biz-Product Workspace

## Mission
Trasformare output R&D in prodotti/servizi con valore economico.

## Workflow
1. Leggi task da `inbox/`
2. Elabora PRD/proposte in `working/`
3. Consegna artefatti in `outbox/`
4. Registra decisioni in `logs/`

## Output standard
- Problema cliente
- Soluzione proposta
- PRD sintetico
- Pricing draft
- Piano go-live
