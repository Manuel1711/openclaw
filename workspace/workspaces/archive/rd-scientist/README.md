# RD-Scientist Workspace

## Mission
Ricerca tecnica, ipotesi, benchmark, validazione scientifica.

## Workflow
1. Leggi task da `inbox/`
2. Esegui analisi in `working/`
3. Consegna output finale in `outbox/`
4. Aggiorna log in `logs/`

## Output standard
- Sintesi tecnica
- Ipotesi + metodologia
- Evidenze
- Rischi
- Next step
