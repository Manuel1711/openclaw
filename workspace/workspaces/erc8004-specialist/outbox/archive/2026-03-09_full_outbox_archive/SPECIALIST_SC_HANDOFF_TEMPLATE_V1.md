# SPECIALIST_SC_HANDOFF_TEMPLATE_V1

## One-line handoff (obbligatorio)
`HANDOFF | TASK=<name> | STATUS=READY_FOR_SC | CODE=<n> | TABLES=<n> | FIGURES=<n> | CPF_DONE=<path>`

## Checklist minima (obbligatoria)
- [ ] Script path (repo):
  - `working/analysis/scripts/...`
- [ ] Tabelle path:
  - `working/results/tables/...`
- [ ] Figure path (PDF):
  - `outbox/figures/...`
- [ ] Technical note path:
  - `outbox/..._TECHNICAL_...md`
- [ ] CPF_DONE path:
  - `outbox/CPF_DONE_...md`
- [ ] Inbox archiviata:
  - `inbox/archive/YYYY-MM-DD_done/TASK_...md`

## Esempio
`HANDOFF | TASK=CLIENT_TEMPORAL_DYNAMICS_2026-03-08 | STATUS=READY_FOR_SC | CODE=1 | TABLES=6 | FIGURES=5 | CPF_DONE=outbox/CPF_DONE_CLIENT_TEMPORAL_DYNAMICS_2026-03-08.md`
