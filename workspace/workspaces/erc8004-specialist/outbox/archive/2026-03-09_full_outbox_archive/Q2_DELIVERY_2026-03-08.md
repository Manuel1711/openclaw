# Q2 Delivery — 2026-03-08

## Method
- Built event-time panel from:
  - `identity_registered.csv` (registration block per agent)
  - `reputation_newfeedback.csv` (feedback block per event)
- For each agent-event: `delta_b = block_feedback - block_registration`.
- Fixed grid: `Δb = 1000` blocks.
- Computed cumulative feedback per agent across Δb buckets.
- Aggregated cross-agent statistics at each Δb: mean, median, p25, p75.

## Limitations
- ETH-only snapshot.
- Strong sparsity: most agents have zero feedback, median remains flat at 0.
- Block-time used as temporal proxy (not wall-clock time).
